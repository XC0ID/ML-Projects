# Nurse Machine Learning Project Report

## 1. Introduction

Healthcare institutions rely heavily on efficient staff management.  
Machine learning can analyze nurse-related data to improve scheduling, performance monitoring, and healthcare outcomes.

---

# 2. Problem Statement

Hospitals must handle complex workforce data including shifts, workload, and staffing efficiency.

Manual analysis is difficult and time-consuming.  
Machine learning models can automate insights and predictions.

---

# 3. Objectives

- Analyze nurse workforce data
- Identify patterns affecting performance
- Build predictive models
- Evaluate model accuracy

---

# 4. Dataset

The dataset contains nurse-related attributes such as:

- Age
- Experience
- Department
- Shift hours
- Workload indicators

Target variable represents predicted category or outcome.

---

# 5. Methodology

### Data Cleaning
Remove missing values and irrelevant features.

### Feature Engineering
Convert categorical variables and scale numeric features.

### Train-Test Split
Dataset divided for training and evaluation.

---

# 6. Models Used

- Logistic Regression
- Decision Tree
- Random Forest

---

# 7. Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score

---

# 8. Results

Random Forest performed best with strong prediction capability.

---

# 9. Applications

- Hospital staff optimization
- Workforce analytics
- Healthcare management systems

---

# 10. Conclusion

Machine learning helps healthcare systems manage staff data effectively and supports data-driven decisions.
